Support — Slot Auto-Tracker

Getting Started

  1. Click the extension icon and paste your User ID from your SecaHub Profile page.
  2. Click Save & Connect.
  3. Navigate to any casino slot game — click the extension icon to show the tracker bar.
  4. Enter your bet, choose a destination (BH / SS / CS), and press Send.

FAQ

The bar doesn't appear when I click the icon
Make sure you're on an http:// or https:// page. The bar cannot inject on browser internal pages like chrome:// or the new tab page.
It says "Not in DB" in yellow
The detected slot isn't in your slots database yet. You can still send data — it will use the name from the URL. To add the slot properly, use the "Submit Slot" form in the Bonus Hunt config.
Where do I find my User ID?
Go to your SecaHub Profile page and click the "Copy Full ID" button. It's a UUID like c9855d5e-b1f3-4db7-af78-...
The wrong slot name is detected
The extension reads the slot name from the casino page URL. Some casinos use IDs instead of names. You can manually type the correct name in the search field on the Bonus Hunt page.
Can I use this on multiple casinos?
Yes — the extension works on any casino website. Click the extension icon on each casino tab to activate the bar.
