Privacy Policy — Slot Auto-Tracker

Last updated: March 17, 2026

1. What This Extension Does

Slot Auto-Tracker is a browser extension that detects which slot game you are playing on online casino websites by reading the page URL. It displays a small top bar so you can submit bet and win amounts to your personal streaming overlay.

2. Data We Collect

  • Page URLs — The extension reads the URL of the active tab to identify the slot game name and provider. URLs are processed locally and are not stored or shared beyond the single database record described below.
  • User-entered data — Bet size, win amount, destination mode (Bonus Hunt / Single Slot / Current Slot), and Super/Extreme bonus flags that you manually enter in the extension bar.
  • User ID — A user-provided identifier stored locally via chrome.storage to associate detected data with your overlay account.

3. How Data Is Used

All collected data is sent exclusively to your own Supabase database instance to update your personal streaming overlay in real time. The extension does not send data to any third-party analytics, advertising, or tracking services.

4. Data Storage

  • Local storage — Your User ID, last detected slot name, and panel preferences are stored locally in the browser using chrome.storage.local.
  • Remote storage — Detected slot name, bet size, win amount, and related flags are stored in a single row (per user) in the detected_slots table of your Supabase project. This row is overwritten with each new detection or submission.

5. Data Sharing

We do not sell, share, or transfer any user data to third parties. Data is only sent to the Supabase project URL configured in the extension.

6. Permissions Explained

  • tabs — Read the active tab URL to detect which slot game is being played.
  • storage — Persist your User ID and panel preferences across browser sessions.
  • host_permissions (<all_urls>) — Inject the input bar on casino game pages. A broad permission is required because casino websites vary widely and cannot be enumerated in advance.

7. Remote Code

The extension does not load or execute any remotely hosted code. All JavaScript runs locally from the extension package. The only network requests are REST API calls to your Supabase database.

8. Children's Privacy

This extension is not intended for use by anyone under 18 years of age. We do not knowingly collect data from minors.

9. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected by the "Last updated" date above.

10. Contact

If you have questions about this privacy policy, contact us at the email listed on the Chrome Web Store developer page.
